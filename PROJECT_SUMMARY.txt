 NEXT STEPS


1. Install Dependencies:
   cd functions && npm install
   cd .. && npm install

2. Configure Environment:
    Get Telegram bot token (@BotFather)
    Get OpenAI API key
    Enable Gmail API in Google Cloud
    Copy .env.example files and fill

3. Run Locally:
   npm run dev              # Frontend :3000
   cd functions && npm run serve  # Backend :5001

4. Deploy:
   firebase deploy --only functions
   vercel deploy            # Or Netlify/Vercel

5. Test:
    Open Telegram bot
    Run /start
    Click /connect
    Authorize Gmail
    Receive email summaries!
